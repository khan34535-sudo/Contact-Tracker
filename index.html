📖 How to Use the Contact Trackers ⬇️
1
➕ Add Contacts
Fill in the form with library/museum details. All fields are optional except Name & Email.
2
✉️ Track Emails
Check the "Email Sent" box when you contact them. The date will auto-record.
3
✏️ Edit Records
Click the pencil icon ✏️ to edit any entry. Save changes instantly.
4
🗑️ Delete Records
Click the trash icon 🗑️ to remove entries (confirmation required).
5
💾 Save Data
Click "Save Data" to export all records as a JSON backup file.
6
🖨️ Print Reports
Click "Print Report" for a professional formatted report of all contacts.
